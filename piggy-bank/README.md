# Piggy Bank 🐷

A delightfully simple iOS app that helps kids track their savings with visual charm and intuitive design.

## Features

- **Account Switcher** - Easy switching between multiple kids' accounts
- **Visual Balance Tracking** - Beautiful charts showing savings over time
- **Transaction History** - Clear record of deposits and withdrawals
- **Kid-Friendly Design** - Large buttons, playful colors, and rounded shapes
- **Themed Interface** - Switchable themes for different visual preferences

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/piggy-bank.git
   ```

2. Open the project in Xcode
   ```bash
   cd piggy-bank
   open PiggyBank.xcodeproj
   ```

3. Build and run the project (⌘+R)

## Architecture

The app follows SwiftUI's MVVM pattern with:
- **Models** - Kid and Transaction data structures
- **Views** - SwiftUI views with single-responsibility design
- **Themes** - Pluggable theme system for visual customization
- **Data** - JSON-based local storage
